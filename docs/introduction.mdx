**codekit** é uma coleção de ferramentas TypeScript para o dia a dia de desenvolvimento: formatação, validação, manipulação de dados, hooks React e helpers para Next.js, tudo com tipagem forte, zero dependências pesadas e defaults pensados para produtos brasileiros (`pt-BR`, CPF/CNPJ, CEP, telefone).

```ts
import { format, is } from 'codekit';

format.cpf('12345678909');        // '123.456.789-09'
format.currency(1500);            // 'R$ 1.500,00'
format.relativeTime(date);        // 'há 3 minutos'

is.email('dev@example.com');      // true
is.cpf('123.456.789-09');         // true
```
