Todos os formatters vivem num namespace único, digite `format.` e o autocomplete lista o catálogo inteiro com as assinaturas reais.

```ts
import { format } from 'codekit';

format.cpf(user.cpf);              // '123.456.789-09'
format.currency(order.total);      // 'R$ 1.500,00'
format.relativeTime(post.date);    // 'há 3 minutos'
format.initials(user.name);        // 'JC'
```

## Null-safety embutida

Toda função do namespace aceita `null | undefined` e devolve o fallback (`'N/A'` por padrão) para valores vazios, sem ternário no call site:

```ts
format.cpf(null);          // 'N/A'
format.phone(undefined);   // 'N/A'
format.list([]);           // 'N/A' (array vazio também é vazio)
format.currency(0);        // 'R$ 0,00', zero é valor legítimo, formata normal
```

## Fallback customizado

`format` é a instância default de `createFormat`, crie a sua por app ou por tela:

```ts
import { createFormat } from 'codekit';

const f = createFormat({ fallback: '-' });

f.cpf(null);     // '-'
f.cpf('123...'); // '123.456.789-09'
```
